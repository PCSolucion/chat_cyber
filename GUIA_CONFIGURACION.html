CHROME-SYNC TWITCH WIDGET // V2.0

01 CONFIGURACIÓN DEL CANAL

Para conectar el widget a tu canal de Twitch, debes editar el archivo de configuración principal.

  1. Navega a la carpeta js/.
  2. Abre el archivo config.js con un editor de texto (VS Code, Notepad++, etc).
  3. Busca la línea que dice TWITCH_CHANNEL: 'nombre_actual'.
  4. Cambia 'nombre_actual' por tu nombre de usuario de Twitch (en minúsculas).
// Ejemplo en js/config.js
const CONFIG = {
  TWITCH_CHANNEL: 'tu_usuario_aqui',
  // ... resto de la configuración
};

02 SISTEMA DE XP Y LOGROS

Este widget guarda el progreso de tus espectadores en un GitHub Gist. Es necesario configurar tus propias credenciales para que los datos se guarden correctamente.

  1. Crea un Gist en gist.github.com (puedes dejarlo vacío con un archivo llamado xp_data.json).
  2. Copia el ID del Gist (es la cadena de caracteres al final de la URL).
  3. Genera un Personal Access Token (classic) en GitHub con permisos de gist.
  4. En el mismo archivo js/config.js, actualiza los campos:
XP_GIST_ID: 'tu_id_de_gist_aqui',
XP_GIST_TOKEN: 'tu_token_de_github_aqui',
ADVERTENCIA: No compartas tu Token con nadie. Si planeas subir este código a un repositorio público, usa variables de entorno o asegúrate de excluir el archivo de configuración.

03 IMPLEMENTACIÓN EN OBS

Una vez configurado, es hora de llevarlo a tu stream:

  1. Carga el widget en un servidor local (puedes usar la extensión "Live Server" de VS Code o simplemente abrir el archivo index.html en tu navegador).
  2. En OBS, añade una nueva fuente de tipo "Navegador" (Browser Source).
  3. En "URL", pega la dirección local (ej: http://127.0.0.1:5500/index.html) o marca "Archivo local" y busca el index.html.
  4. Configura el tamaño:
    • Ancho: 450 (ajustable)
    • Alto: 800 (ajustable)
  5. ¡Listo! El chat empezará a aparecer cuando alguien escriba.

04 COMANDOS DEL CHAT

Tus espectadores pueden interactuar con el widget usando estos comandos:

Comando Descripción
!level Muestra tu nivel actual y XP necesario.
!stats Resumen de tus estadísticas en el canal.
!top Muestra el ranking de los mejores usuarios.
!logros Lista tus logros desbloqueados.
